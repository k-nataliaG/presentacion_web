REDES:

TIPOS DE REDES INFORMÁTICAS

Red de Área Local (LAN)

La Red de Área Local es una red que se limita a una zona geográfica reducida, como una oficina, un edificio o una escuela. Esta red es administrada por una sola organización y se utiliza para compartir recursos como impresoras, archivos, entre otros. En esta red, los dispositivos están conectados mediante cables o redes inalámbricas. Las LAN pueden ser cableadas, es decir, que los dispositivos se conectan mediante un cable Ethernet, o inalámbricas, que utilizan la tecnología Wi-Fi para conectar los dispositivos a la red.

Red de Área Amplia (WAN)

La Red de Área Amplia una zona geográfica mayor que una LAN, como una ciudad, un país o incluso abarca varios países. Estas redes son utilizadas para conectar múltiples LAN y permitir la comunicación entre ellas. Las WAN son administradas por diferentes organizaciones y suelen utilizar diferentes tecnologías de conexión, como líneas telefónicas, satélites, cables submarinos, entre otros.

Red de Área Metropolitana (MAN)

La Red de Área Metropolitana es una red que cubre una zona geográfica mayor que una LAN, pero menor que una WAN. Esta red se utiliza para conectar diferentes LAN ubicadas en una misma ciudad o zona metropolitana. Las MAN son administradas por diferentes organizaciones y suelen utilizar tecnologías como fibra óptica, cableado Ethernet y Wi-Fi.

Red de Área de Almacenamiento (SAN)

La Red de Área de Almacenamiento se utiliza para conectar dispositivos de almacenamiento, como discos duros, a diferentes servidores. Esta red permite el acceso a los datos almacenados de manera más rápida y eficiente. Las SAN utilizan tecnologías de conexión de alta velocidad, como fibra óptica, para garantizar un acceso rápido y seguro a los datos almacenados.

CARACTERÍSTICAS DE LAS REDES INFORMÁTICAS

Topología

La topología de una red se refiere a la forma en que están conectados los dispositivos. Las topologías más comunes son la topología en estrella, en bus y en anillo.

Protocolos

Los protocolos son un conjunto de reglas que definen la forma en que los dispositivos se comunican entre sí. Los protocolos más comunes son TCP/IP, HTTP, FTP y SMTP.

Ancho de banda

El ancho de banda se refiere a la cantidad de datos que pueden ser transmitidos por segundo en una red. Un ancho de banda más alto significa que la red puede transmitir más datos en menos tiempo.

Seguridad

La seguridad es una característica importante en cualquier red informática. Se deben implementar medidas de seguridad para proteger los datos y dispositivos de posibles amenazas externas, como virus, hackers y malware.

TIPOS DE LICENCIAS

Tipos de Licencias Teniendo en cuenta las especificaciones del software debo aclarar que los servidores Linux en sí mismos no ofrecen licencias, sino que la mayoría de las distribuciones de Linux son software libre y están disponibles para su uso y distribución de forma gratuita bajo una licencia de código abierto, como la Licencia Pública General de GNU (GNU GPL).

Sin embargo, hay algunas excepciones en las que se puede requerir una licencia para utilizar ciertos programas o aplicaciones en el servidor. Algunos ejemplos son:

• Si estás utilizando software comercial en el servidor, es posible que necesites comprar una licencia para utilizarlo legalmente.

• Algunas aplicaciones de software libre pueden tener licencias restrictivas, que limitan su uso a ciertos propósitos o exigen que cualquier software derivado también esté disponible bajo una licencia de código abierto.

Por lo tanto, es importante revisar las licencias de los programas y aplicaciones que planeas utilizar en tu servidor Linux, para asegurarte de que no infringes ningún derecho de autor o licencia. Generalmente, las licencias de software se pueden encontrar en la documentación o el sitio web del desarrollador del software.

Licencia Pública General de GNU (GNU GPL):

es una licencia de código abierto que permite a los usuarios utilizar, modificar y distribuir el software siempre y cuando cualquier modificación también esté disponible bajo la misma licencia.

Licencia Pública General Reducida de GNU (GNU LGPL):

es una licencia de código abierto similar a la GNU GPL, pero con algunas restricciones adicionales en cuanto a la distribución de bibliotecas de software.

Licencia MIT:

es una licencia de código abierto muy permisiva que permite a los usuarios utilizar, modificar y distribuir el software, incluso en aplicaciones comerciales, siempre y cuando se incluya el aviso de derechos de autor y la licencia original.

Licencia Apache:

es una licencia de código abierto similar a la Licencia MIT, pero con algunos términos adicionales relacionados con las patentes y la responsabilidad legal.

Licencia BSD:

es una licencia de código abierto muy permisiva que permite a los usuarios utilizar, modificar y distribuir el software, incluso en aplicaciones comerciales, siempre y cuando se incluya el aviso de derechos de autor y la licencia original.


Es importante tener en cuenta que estas son solo algunas de las licencias comunes que se utilizan en el software que se ejecuta en servidores Ubuntu, y que cada programa o aplicación puede tener su propia licencia específica. Por lo tanto, es importante revisar las licencias de cada programa que planeas utilizar para asegurarte de cumplir con sus términos.
